How is this being calculated?
This submission score will always generate a weighted value based on the scoring table you've selected. The value will be displayed as a percentage on the submission record.
Output Value
Sum of (scoring table output * weightage) / 100

What is a scoring table?
A scoring table is used during the creation of a submission score rule. The scores in these tables will be used as inputs for a weighted average calculation for the submission score. It is recommended to provide a score from a scale of 1-100, but this can vary by carrier.
