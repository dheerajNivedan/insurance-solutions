# Appian Brand Colors

When building Appian prototypes, always use these brand colors:

- **Primary Accent (Appian Blue):** `#2322F0`
  - Use for: active tabs, links, nav highlights, active borders, primary buttons, chart accent color, avatar backgrounds
  - Tailwind custom: use `style={{ color: '#2322F0' }}` or `style={{ backgroundColor: '#2322F0' }}` when Tailwind classes don't match
  - Do NOT use `blue-500`, `blue-600`, `blue-700`, or `blue-900` as substitutes — always use `#2322F0`

- **Positive (Green):** Tailwind `green-700` / `#15803d`
- **Negative (Red):** Tailwind `red-700` / `#b91c1c`
- **Secondary (Gray):** Tailwind `gray-700` / `#374151`
- **Standard (Dark):** Tailwind `gray-900` / `#111827`
